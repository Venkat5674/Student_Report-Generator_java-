# iText Library Download Instructions

To enable proper PDF generation, follow these steps to download the iText library:

1. Visit https://github.com/itext/itextpdf/releases
2. Download the latest release of `itextpdf-x.x.x.jar` (e.g., `itextpdf-5.5.13.3.jar`)
3. Place the downloaded JAR file in this folder

Once the JAR file is in place, the application will automatically use it for proper PDF generation.
